# final_project
# ⚾ 잠실 야구장 좌석 가이드: 당신의 완벽한 직관을 위한 선택

> **"오늘 경기 어디서 보지? 비 오면 어쩌지? 뭐 챙겨가야 하더라?"**
> 잠실 야구장 직관을 계획하는 팬들을 위한 올인원(All-in-One) 가이드 서비스입니다. 최고의 관람 경험을 위해 좌석 추천부터 날씨, 준비물까지 한 번에 해결해 드립니다.

---

## 🚀 주요 기능 (Key Features)

### 1. 💺 맞춤형 좌석 추천
* **성향별 추천:** 응원 열기를 느끼고 싶은 팬(오렌지석), 편하게 음식을 먹으며 보고 싶은 팬(테이블석), 경기 흐름을 한눈에 보고 싶은 팬(네이비석) 등 유저 성향에 맞는 최적의 구역을 추천합니다.
* **시야 미리보기:** 선택한 좌석 구역에서 실제로 보이는 경기장 시야 정보를 제공하여 예매 실패 확률을 줄입니다.

### 2. ☀️ 실시간 날씨 확인
* **경기 시간대 날씨:** 단순 일일 날씨가 아닌, 경기 시작 및 종료 시간대의 기온, 강수 확률, 미세먼지 정보를 실시간으로 연동하여 제공합니다.
* **우천 취소 예측:** 강수량 데이터를 기반으로 직관 당일 우천 취소 가능성을 사전에 안내합니다.

### 3. 🎒 스마트 준비물 체크리스트
* **시즌별 필수템:** 봄/가을(외투, 무릎담요), 여름(선크림, 손선풍기) 등 계절에 맞는 준비물을 가이드합니다.
* **날씨 연동 추천:** 비 소식이 있을 경우 '우의/우산', 자외선이 강할 경우 '모자/선글라스'를 체크리스트에 자동으로 추가해 줍니다.
